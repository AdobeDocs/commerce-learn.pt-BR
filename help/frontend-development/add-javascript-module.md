## Etapas

- Criar um módulo
- Crie um `requirejs-config.js` e um arquivo de módulo JavaScript
- Criar atualização de layout para adicionar um modelo que ative o módulo JS
- Criar arquivo de modelo
- Adicione o módulo e teste-o
