# Aumente o desempenho da vitrine eletrônica com o Edge Delivery

Descubra como as vitrines do Adobe Commerce alimentadas pela Edge Delivery Services alcançam velocidade e resiliência excepcionais. Saiba como a renderização inteligente de página, o armazenamento em cache persistente e a arquitetura CDN melhoram as pontuações do Lighthouse, o Core Web Vitals e o SEO prontos para uso.

## Para quem é este vídeo?

* Desenvolvedor front-end
* Engenheiro de desempenho do site
* Gerenciador de comércio eletrônico

## Conteúdo de vídeo

* Saiba como a arquitetura do Edge Delivery permite que as vitrines tenham pontuação acima de 90 no Lighthouse e transmitam no Core Web Vitals.
* Descubra como a entrega e o armazenamento em cache alimentados por CDN melhoram as taxas de SEO, tráfego e conversão sem esforço extra.
