# Arquitetura do Adobe Commerce as a Cloud Service

O Adobe Commerce as a Cloud Service oferece uma plataforma de hospedagem segura e automatizada com recursos de autoatendimento para gerenciar aplicativos de comércio. Os principais recursos incluem gerenciamento de ativos por meio do Adobe Experience Manager, cobertura de API abrangente com APIs GraphQL e REST e serviços inteligentes da Adobe Merchandising. A plataforma oferece suporte a uma arquitetura totalmente headless, permitindo que os desenvolvedores usem várias estruturas front-end, como o React e o Angular. Ele também fornece uma loja headless moderna com criação baseada em documentos e testes A/B nativos. Além disso, o Adobe Commerce se integra à Adobe Experience Cloud para experiências de comércio personalizadas e oferece ferramentas abrangentes para desenvolvedores, incluindo o Adobe Developer App Builder e a malha de API.
