# Introdução à ferramenta Migração de dados em massa

Prepare o Docker e acesse a Ferramenta de Migração de Dados em Massa para migrar dados do Adobe Commerce para [!DNL Adobe Commerce as a Cloud Service] ([!DNL ACCS]).

## Para quem é este vídeo?

* Arquiteto de soluções
* Engenheiro de DevOps
* Desenvolvedor de backend

## Conteúdo de vídeo

* Saiba mais sobre os recursos e as limitações da Ferramenta de migração de dados em massa.
* Confirme se o Docker está instalado e se o mecanismo do Docker está em execução antes de iniciar.
* Extraia o pacote de ferramentas e defina as permissões de pasta corretas.
* Revise o arquivo `.env.example` configurado nos próximos vídeos desta série.
