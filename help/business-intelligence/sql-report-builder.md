# SQL REPORT BUILDER

O Adobe Commerce Intelligence SQL Report Builder permite consultar o data warehouse, revisar resultados e transformá-los em gráficos. Ao criar relatórios com SQL, você não espera nos ciclos de atualização para refinar as colunas criadas. Se algo parecer estranho, edite e execute novamente a query até que a saída corresponda ao que você espera.

>[!IMPORTANT]
>
> Você precisa de permissões de Administrador para usar o SQL Report Builder.

>[!TIP]
>
> Se você é novo no Commerce Intelligence, o [guia do usuário do Adobe Commerce Intelligence](https://experienceleague.adobe.com/docs/commerce-business-intelligence/mbi/guide-overview.html?lang=pt-BR){target="_blank"} explica como agregar, analisar e visualizar dados para obter insights comerciais mais sólidos.
