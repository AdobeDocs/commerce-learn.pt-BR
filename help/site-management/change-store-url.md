# Alterar uma URL de loja

Cada site em uma instalação do Adobe Commerce ou do Magento Open Source tem um URL base atribuído à loja e outro URL atribuído ao administrador. Você pode seguir algumas etapas simples para alterar o URL de base da sua loja.

* Administradores de armazenamento
