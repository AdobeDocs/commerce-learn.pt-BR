# Instalação e configuração

Saiba como obter observabilidade de ponta a ponta em aplicativos Adobe Commerce e personalizados usando telemetria padronizada e correlacionada. Este vídeo explica por que a correlação e a padronização são importantes, apresenta o Módulo de observação da Commerce e a Biblioteca de telemetria IO em tempo real. Saiba como essas ferramentas permitem monitoramento, registro e rastreamento unificados para um fluxo operacional contínuo.

## Para quem é este vídeo?

* Desenvolvedor do Adobe Commerce
* Arquiteto de soluções
* Gerente técnico de produto

## Conteúdo de vídeo

* Padronizar e correlacionar telemetria para total observabilidade
* Usar o módulo de observação Commerce para sinais automatizados
* Estender monitoramento com IO Live Telemetry Library para aplicativos
