# Visão geral da capacidade de observação

Saiba por que a observabilidade centralizada é essencial para as arquiteturas modernas do Adobe Commerce. À medida que a extensibilidade evolui para um modelo combinável com aplicativos independentes, webhooks e extensões de interface do usuário, os comerciantes enfrentam o desafio de rastrear fluxos de execução complexos. Saiba como uma camada de observabilidade unificada correlaciona sinais entre serviços, permitindo uma otimização proativa e garantindo experiências de finalização rápidas e confiáveis.

## Para quem é este vídeo?

* Desenvolvedor do Adobe Commerce
* Arquiteto de soluções
* Gerente técnico de produto

## Conteúdo de vídeo

* O modelo combinável adiciona complexidade: aplicativos e webhooks independentes não têm um único contexto.
* A capacidade de observação centralizada é fundamental: correlacione registros e métricas em todos os componentes.
* Aumentar o desempenho da finalização: identifique gargalos rapidamente para manter as compras tranquilas.
